Afinal, o que é uma linha de comando?

  • É interface de usuário onde a interação com um programa de computador é feita por sucessivas linhas de texto (linhas de comando).

  • Qualquer programa pode ter uma interface como essa.

  • Essa apresentação foi feita por meio de ferramentas que só tem uma interface via linha de comando.
